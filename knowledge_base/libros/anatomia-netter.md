# Atlas de Anatomía Humana - Netter

## Información del Libro
- **Edición**: 7ma (2019)
- **Editorial**: Elsevier
- **Uso**: Atlas de referencia complementario en todas las universidades
- **Nivel**: Referencia visual universitaria

## Contenido Principal
- Más de 600 ilustraciones anatómicas de alta calidad
- Organizado por regiones: cabeza/cuello, dorso/médula, tórax, abdomen, pelvis, miembros
- Incluye imágenes de superficie, secciones transversales
- Correlaciones clínicas en cada sección
- Tablas de músculos con origen, inserción, acción e inervación

## Relevancia para Kinesiología
- Referencia visual definitiva para identificar estructuras
- Complemento indispensable del Latarjet
- Tablas musculares útiles para evaluación y tratamiento
- Inervación segmentaria para diagnóstico diferencial
