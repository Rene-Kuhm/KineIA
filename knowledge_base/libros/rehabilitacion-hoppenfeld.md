# Exploración Física - Hoppenfeld

## Contenido Principal

### Evaluación por Región
- **Columna Cervical**: Inspección, palpación, arcos de movimiento, tests especiales (Spurling, compresión/distracción)
- **Hombro**: Test de Neer, Hawkins, Jobe, Speed, Yergason, cajón anterior/posterior
- **Codo**: Tests de epicondilitis lateral/medial, estabilidad ligamentaria
- **Muñeca/Mano**: Phalen, Tinel, Finkelstein, Allen
- **Columna Lumbar**: Lasègue (SLR), Bragard, test de Schober, tests de estabilidad
- **Cadera**: Thomas, Trendelenburg, FABER/Patrick, Ober
- **Rodilla**: Cajón anterior/posterior, Lachman, McMurray, Apley, tests meniscales
- **Tobillo/Pie**: Cajón anterior, Thompson, inversión forzada

### Metodología de Exploración
1. Inspección (estática y dinámica)
2. Palpación ósea y de tejidos blandos
3. Arcos de movimiento (activo y pasivo)
4. Evaluación neurológica (fuerza, sensibilidad, reflejos)
5. Tests especiales

## Relevancia para Kinesiología
- Manual de exploración física más usado en formación
- Tests ortopédicos validados para diagnóstico diferencial kinésico
- Sistemática de evaluación por región
